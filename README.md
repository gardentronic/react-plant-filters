## Installation & Usage

```sh
npm install react-plant-filters --save
```

### Include the Component

```js
import React from 'react'
import { PlantFilters } from 'react-plant-filters'

class Component extends React.Component {

  render() {
    return <PlantFilters />
  }
}
```
You can import  respectively.
